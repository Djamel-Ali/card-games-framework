# card-games-framework
The concrete goal is to realize applications allowing to play several card games.

in root ( card-games-framework directory) :

* **Compilation :** ***`$ make`***

* **Execution :** ***`$ ./main`***

* **Cleaning :** ***`$ make clean`***
